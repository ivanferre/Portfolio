Pictures in HTML

February 24, 2023

I'm going to demonstrate some responsive techniques with images in HTML5.

I have used a photo by kabita Darlami on Unsplash, and the crop function from the amazing and free I love IMG website. A must if you use images in your websites.

Responsive Images

I'm following this source of knowledge:

  • Responsive Images 101, Part 1: Definitions, by Jason Grigsby.
  • Images with Bootstrap 4.
  • Responsive Images (Mozilla).
  • Responsive Images (Bootstrap).

And I plan to study as well Responsive Images 101, Part 8: CSS Images, by Jason Grigsby, too.

Including the original image in the website is the easy part. The tricky part is to display it appropriately. Let's have some examples.

We attempt to solve the Art Direction Problem by specifying two different images in a picture element. If the screen (actually, the viewport) is narrow enough, you may see how the image displayed below changes.

The picture is displayed with two different enhancements. First, we set a margin on both top and bottom as explained in the Bootstrap Spacing techniques page. Second, we round the corners by using the rounded class as shown in the Borders page.
