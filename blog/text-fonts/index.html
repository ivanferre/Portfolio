Text Formatting: Fonts

February 26, 2023

There are a number of considerations on choosing the typefonts. Aesthetics, since they give personality to the texts and pages; and technical, because accessibility and time to load must be taken into consideration.

To use a specific font in a page, we must perform two steps:

  1. Import the font from the source. This may be done in turn in two places:
    • The HTML page, With the link tag, exactly the same way a .css file would be included, from the header or the beginning of the body. However, I prefer to keep the HTML pages only for the content and structure of the document.
    • The CSS file, which looks the best place to me since the style sheet is meant for exactly this: defining the looks of the document. In this case, we must use the @import statement.
  2. Use the font in the appropriate styles for body and headers.

Google Fonts, Adobe Fonts, Font Squirrel, and Da Font are four font repositories to consider.
